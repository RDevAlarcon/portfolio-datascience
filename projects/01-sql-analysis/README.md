# SQL Analysis

Consultas SQL orientadas a analisis de datos relacionales. El ejercicio trabaja joins, filtros, agrupaciones y extraccion de informacion util desde una base transaccional.

## Archivo principal

- `customer_orders_joins.sql`

## Habilidades mostradas

- Joins entre multiples tablas
- Consultas con condiciones y ordenamiento
- Agregaciones para analisis
- Pensamiento relacional aplicado a datos de negocio
