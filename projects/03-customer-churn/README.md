# Customer Churn Prediction

Modelo de clasificacion para predecir fuga de clientes en un contexto de telecomunicaciones. El proyecto muestra preparacion de datos, entrenamiento y evaluacion de modelos.

## Archivo principal

- `customer_churn_prediction.ipynb`

## Dataset

- `Telco-Customer-Churn.xlsx`

## Habilidades mostradas

- Preprocesamiento de variables categoricas y numericas
- Clasificacion supervisada
- Evaluacion con metricas de desempeno
- Interpretacion de factores asociados al churn
