# Flight Price Prediction

Modelo predictivo para estimar precios de vuelos usando datos de clase economica y business. El proyecto combina datasets, transforma variables y evalua modelos de regresion.

## Archivo principal

- `flight_price_prediction.ipynb`

## Datasets

- `economy.xlsx`
- `business.xlsx`

## Habilidades mostradas

- Union y preparacion de datasets
- Feature engineering
- Modelos de regresion
- Comparacion de desempeno predictivo
