# Tableau Sales Dashboard

Dashboard comercial desarrollado en Tableau para analizar ventas de retail. El archivo empaquetado permite revisar visualizaciones y KPIs desde Tableau Desktop o Tableau Public.

## Archivo principal

- `sales_dashboard.twbx`

## Dataset

- `Sales_Retail.csv`

## Habilidades mostradas

- Visualizacion de KPIs
- Analisis comercial
- Construccion de dashboard
- Comunicacion visual de resultados
