# House Price Prediction

Proyecto de regresion para estimar precios de viviendas. El notebook trabaja exploracion de variables, preparacion de datos y evaluacion predictiva.

## Archivo principal

- `house_price_prediction.ipynb`

## Dataset

- `house_data.xlsx`

## Habilidades mostradas

- Analisis exploratorio
- Regresion supervisada
- Seleccion y transformacion de variables
- Evaluacion de errores de prediccion
