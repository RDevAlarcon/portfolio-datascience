# Customer Segmentation

Segmentacion de clientes a partir de datos transaccionales de retail. El objetivo es identificar grupos de clientes con patrones de compra similares.

## Archivo principal

- `customer_segmentation.ipynb`

## Dataset

- `Retail_Invoices.xlsx`

## Habilidades mostradas

- Preparacion de datos transaccionales
- Clustering
- Analisis de segmentos
- Interpretacion comercial de resultados
